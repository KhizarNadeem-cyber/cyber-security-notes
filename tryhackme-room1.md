📌 Overview

Defensive Security ka focus systems ko protect karna, attacks detect karna aur incidents ka response dena hota hai.


 🧠 Key Concepts

* **Blue Team** – System ko secure karta hai
* **SOC (Security Operations Center)** – Monitoring aur detection
* **Log Analysis** – Suspicious activities identify karna
* **Incident Response** – Attack hone par response dena


 🧰 Tools Used

* SIEM Tools
* Firewalls
* Antivirus Software


🎯 Objectives

* Threat detection
* Incident handling
* System protection


🧠 Key Learnings

* Defensive Security proactive aur reactive dono hoti hai
* Monitoring aur logging bohat important hain
* Quick response attacks ko control karta hai


🏁 Conclusion

Defensive Security organizations ko cyber attacks se bachane ka core part hai aur ek strong security foundation provide karta hai.
